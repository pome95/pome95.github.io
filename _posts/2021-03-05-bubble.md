## 버블 정렬 (Bubble Sort)
> 두 인접한 데이터를 비교해서 정렬하는 알고리즘

## 핵심 코드
버블 정렬은 다음과 같은 코드를 베이스로 구현이 가능하다.

```python
for index in range(데이터 길이 -1):
    for index2 in range(데이터 길이 - index -1):
        if 앞 데이터 > 뒤 데이터:
            swap(앞 데이터, 뒤 데이터)
```
